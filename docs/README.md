## Overview

`sp-treeview-v2` is an Angular library that provides a reusable, feature-rich treeview component for displaying hierarchical data with infinite nesting levels.

## Key Features

- Infinite nesting depth
- Lazy loading with loading indicators
- Three selection modes: none, radio, checkbox
- Search/filter across the entire tree
- Add child and delete node operations
- Three UI variants: standard, dropdown, overlay
- Custom node templates via `ng-template`
- Per-node configuration overrides
- Built on Angular Material
